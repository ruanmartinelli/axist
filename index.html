axist

Minimal (5kb) classless CSS library.

GitHub

Usage

Install via npm:

npm i axist

Or import in your HTML:

<link rel="stylesheet" href="https://unpkg.com/axist@latest/dist/axist.min.css" />

No classes needed — it styles plain HTML elements, like the ones on this page.
